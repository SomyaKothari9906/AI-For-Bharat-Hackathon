# Project Title:
Fake Welfare Beneficiary Detection using AI

## 1. Problem Statement
Government welfare schemes often suffer from fake and duplicate beneficiaries.
Manual verification is slow and inefficient.

## 2. Objective
Build an AI-based system to detect duplicate and fake beneficiaries automatically.

## 3. Functional Requirements
- Upload beneficiary dataset (CSV format)
- Detect duplicate records using AI
- Identify suspicious patterns
- Generate fraud probability score
- Admin dashboard to view flagged records

## 4. Non-Functional Requirements
- Fast processing
- Secure data handling
- Scalable architecture

## 5. Target Users
- Government officials
- Welfare department administrators
